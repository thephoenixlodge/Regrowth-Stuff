Millénaire : 日本建築

村莊中心

Shuudannoujou : 大農場
農業村莊的中心，農夫和他的妻子住在這裡。
村民們會把農作物、木、石之類的物資帶到
這裡。

Bousai : 堡壘 
領主與他的妻子住在這裡，可以透過升級城牆
和塔樓來成為一座強大的堡壘。

Kumonjo : 帝國指揮部
日本會在村莊中心建立龐大的帝國指揮部。
監督者和他的妻子居住在這裡，日夜守護著村莊。
村莊所有的建設計畫都在這裡製作，額外的計畫
和建議也是如此。

住宅

Shoko : 檔案館 
這座建築記載著村莊的村民們的所有資料。

Tansetsu : 軍匠鋪 
軍匠舖可以製造所有日本軍隊的武器和鎧甲。
他們可以製造太刀和致命的和弓。
他們也做出各式的日本鎧甲。
這裡需要一位工匠來製造武器和他的妻子來
販售武器。

Tansetsu : 鐵匠鋪 
這裡是製造和販賣鐵製工具的地方。
鐵匠舖製造的工具能幫助人們更快地完成工作。
玩家也可以在這裡購買工具。

Shugi Ie : 鍛造屋
生產並銷售用米做成的紙，紙牆和木框架
也會在主建築內出售。

Gyogyo and Tsuribori : 漁場 
漁民將他的時間都花在這裡替村莊捕魚。

Kodachi : 小樹林 
日本樵夫會在這裡砍樹。

Shijin : 旅館 
在旅行商人從另一個村莊運來貨物之後，他們
會到這裡來休息。

Daidokoro : 廚房 
這裡是料理並煮熟烏龍麵的地方。
此外這裡還會販售雞肉和魚。

Kikori Koya : 樵夫的房屋
這是樵夫和他的妻子居住的房子。
樵夫會砍下樹木，妻子將它帶到村莊中心。

Bourou : 瞭望塔 
瞭望塔大多都是建在村莊的邊緣，但看起來
更像是個裝飾用的建築物。
但興建這個建築的主要原因是為了讓守衛能
夠看見從遠方接近的敵人。

Suiden : 水稻田 
村民們會在這裡種植水稻，它是由農民和婦女
來種植。婦女們會將這裡的稻米帶到村莊
中心去，而玩家可以到那裡購買稻米。

Buttou : 寶塔 
僧人住在這裡，而村民們會到這裡來參拜。

Noufu Ie : 平民的房屋
這裡是農民和他妻子生活的家。

Saisekijou : 採石場 
這裡是礦工和他妻子生活的家。
這裡會開採出石頭，並送到村莊中心去。

Heisha : 守衛的房屋
這裡是守衛(或武士)和他妻子生活的家。

Noujou Nojo : 小麥農場 
這裡是農夫和他妻子生活的家。
他會在種植並收成小麥。

Ike : 池塘 
池塘是一個很好的裝飾建築，
是個不錯的散步地點。

Torii : 鳥居 
一個美麗的門狀建築，這裡的村民
會前來參拜祈福。

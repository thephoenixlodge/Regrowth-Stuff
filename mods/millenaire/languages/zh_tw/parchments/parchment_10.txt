Millénaire: 馬雅建築

村莊中心

Muul : 瑪雅金字塔
瑪雅人建造了數百種類型的金字塔，
從最見的階梯金字塔到平台金字塔都有
而且他們會在金字塔建築之上建造他們的
宮殿、廟宇、祭壇等等的。

K'uhul Jalab : 皇家球場
皇家球場是個殘酷的運動場地，佔地約一個
足球場大，有兩座26英尺高的雄偉牆壁。
每一隊的七名戰鬥員都要想盡辦法取得一個小
橡皮球，讓它通過離地面23英尺高的小石環，
而且據說他們甚至不能用手腳碰球。
輸掉的球隊有時候會被獻祭。

Kan Cha'an : 天文台
古瑪雅人建造天文台，並以如太陽和昂宿星團
等天體的運動來校准他們最重要的建築物。
瑪雅的天文學家們找到了太陽的位置、月亮的
北部和南部的偏角，還有金星的軌道位置。
他們利用房間內的太陽照射在門口之陰影的投
影角度來計算夏至和冬至的到來。

住宅

Maya Otoch : 瑪雅住宅
這些建築一開始是由簡單的泥屋、木屋和石屋
開始，但隨著馬雅文明的發展，它們會被石頭
建築所取代。傾斜的茅草屋頂小屋也給了他們
除了石頭圓頂之外的不同風格外貌。

B'aat Winik Nah : 樵夫的房屋
這是為村莊的發展提供木材的樵夫們的居所。

Tunich In-ti'a'al : 採石場
瑪雅村莊的成長需要極大量的岩石，其中最常見
的就是石灰石，這種石頭在村莊有限範圍內的礦
山淺層可以找到。

Kob Took : 工匠的房屋
這是常常被工匠們裝飾的家。部落所需的各種物
品經常由這裡來製造。

Ah Tz'on Si' Took : 獵人鍛造舖
這是用來製造和儲存各種武器的地方，這些武器
常用在瑪雅人日常的狩獵和戰爭裡。

Uxul Otoch : 雕刻師的房屋
這是村莊裡雕刻師所使用的家。雕刻師們經常把
把他們的作品放在家中以便進行創作。

Maize Makan : 玉米農場
瑪雅文明以玉米作為主食，並且將其種植在瑪雅
村莊周圍。一些發展到人口數十萬的城市更加
需要大量種植玉米。

Ah Kax Makan : 養雞場
瑪雅人養殖雞隻主要是為了作為肉類，而這些肉
常做成瑪雅著名的玉米粉蒸肉。

Chanan Nah : 守衛的房屋
瑪雅戰士在很多情況下只是由一些樸實的村民
們擔任，而他們事後也能回到自己的崗位。
有些專業的戰士經通於狩獵和戰爭的藝術，
這些戰士的家裡常常擺放著他們在冒險中取得
的戰利品。

Ah Waay Su'uk : 巫師庭院
瑪雅巫師和他們妻子的居所，傾注著
巫師的信仰。

K'as Koloh-che'/K'as Otoch : 
土匪哨站/土匪的房屋
這是那些不受村莊所歡迎的瑪雅人的居所。
這些地方通常由一兩間小棚屋構成，然後有
時被驅逐的瑪雅人會發現無人居住的廢墟，
將其重建成自己的家。
建議避開這個地區，因為他們會為了保護自己
的家園而殺人。

無人居住的建築物 :

Jalab : 球場
球場通常是用來玩球類遊戲，但有時候會玩命。
瑪雅的中心城區是出於宗教原因而建。
幾乎每個瑪雅城市都有個球場來玩球類遊戲
Pok-A-Tok。
Pok-A-Tok遊戲通常是宗教儀式所需的一部份。

K'aax : 小樹林
是一個為村莊提供木材的主要供應點。
瑪雅人會找個樹木集中區來砍伐木材。
一但樹木被砍伐殆盡，他們會遷村去尋找新
的地點。

Hu'um-Noh : 圖書館
瑪雅圖書館只是個用來記載村莊的成就
和重要人物的地方罷了。

K'u-nah : 寺廟
瑪雅人的寺廟往往都是很複雜的。
這裡是進行宗教活動的地方，常常住著
皇家祭司。

K'eyem : 祭壇
瑪雅人的祭品從動物到人類都有，這些獻祭
過程中最壯觀的地方就是受害者的鮮血會覆
蓋著整個祭壇。

Chak : 麵包店
瑪雅人會到這裡來為部落將食物烹煮成餐點。
玉米薄餅和雞肉玉米薄餅會在麵包店裡製作
出來，與瑪雅美食如可可之類的一起享受放鬆
與休息。

K'in Pak : 瑪雅日曆
古代瑪雅人發明了十分卓越的精確而複雜的日曆
瑪雅日曆使用三種並行卻不同的測定系統，長曆
法、卓爾金歷(聖歷)和哈布(民用歷)。其中，只
有哈布與一年時間的長短有關係。

Ch'en : 洞穴
一個神祕的古瑪雅洞穴。

Pwes : 蓄水池
蓄水池是井的延伸物，瑪雅人用它來收集並儲存
雨水以供村莊使用。
瑪雅人經常會在蓄水池的開口放置方塊將它封閉
，只留下中間一個洞來取水。

Laak' Otoch / Laak'Noh : 訪客房屋與客棧
這些只是簡單的空房屋，提供給Pok-A-Tok遊戲
的隊伍住宿。

Cha'an Ka'nal : 瞭望塔
這些塔樓用來觀看密不透光的叢林。
會去使用這個建築可能有著很多理由，從盡可能
地去尋找獵物，到利用附近當地人的狼煙來看守。

Hach Noh Otoch / Hach Noh Nah / Yalam Otoch / Ni'Nah :
超大住宅/住宅/平房/閣樓
要是村莊中心附近的空間過於稀少時，瑪雅人
就會蓋這種結構的建築。
瑪雅人會合併他們的家庭並共用他們的房間，
以此來節省空間。
此舉往往會創出比宏偉建築更高的房屋。

Sa'at Muul : 被遺忘的金字塔
在瑪雅的歷史紀錄上，曾有幾個村莊在一夜
之間突然消失得無影無蹤。
關於這些失蹤的謎團還沒有個合理的解釋。
有時這些失蹤村子的廢墟會存留下來，而且
據說這些廢墟之中藏有寶藏。

Sa'at Maya Maize Makan : 被遺忘的瑪雅玉米農場
有時瑪雅人會因為很多原因而不得不離開
自己的家園。有時留下來的建築會有各種
各樣的東西，那些都是原本的居民留下的。
如果你碰巧遇到一個被遺忘的瑪雅玉米農場，
要記得找找看有沒有遺留下來的玉米。

Sa'at Maya Otoch : 被遺忘的瑪雅房屋
另一種被遺忘的瑪雅住宅。這種住宅是
大型建築，好像是在遙遠的過去就被
遺忘了。在被遺棄之前，這裡應該住著
一個大家庭。

Sa'at Tunich In-ti'a'al : 被遺忘的石頭礦井
這個被遺忘的石頭礦井，很明顯的廢棄了很久。
有一個無法被理解的謎就是，這裡的礦井竟然
還留有大量的礦石。
